# Demo Script

Target length: 2 to 3 minutes.

## 0:00 - 0:20

Show the workbench and state the problem: builders need fast security and Mantle deployment feedback before they submit or deploy contracts.

## 0:20 - 0:50

Connect the wallet on Mantle Sepolia. Load the demo contract and point out the risky withdraw flow.

## 0:50 - 1:30

Run the audit. Show the generated risk score, high-risk reentrancy finding, tx.origin issue, recommended fixes, and gas optimization items.

## 1:30 - 2:10

Submit the source hash and report hash on-chain. Show the transaction hash and Proof of Audit token ID.

## 2:10 - 2:40

Open the report page. Explain that the complete code and report stay off-chain by default, while verifiable hashes and proof live on Mantle Sepolia.

## 2:40 - 3:00

Close with track fit: AI DevTools for Mantle builders, focused on smart gas optimisation and Mantle-specific audit assistance.
